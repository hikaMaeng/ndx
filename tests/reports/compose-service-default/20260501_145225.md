## Failures

The first compose startup attempt exposed a configuration gap: `ndxserver
--cwd /workspace` looked for settings under the empty workspace volume and
entered the settings wizard instead of reaching the listen step.

Final fix: the default compose command starts `ndxserver --mock` for service
wiring verification. It does not copy repository settings into `/home/.ndx`.
